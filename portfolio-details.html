Teamhub

Project information

  • Demo URL: Teamhub
  • Teamhub is a network based collaboration tool.
  • Since Network markets are in tree-like structure, this software is tailored to cater for these needs.
  • Built with technologies such as Asp.net core 3.1, Angular 10. Also includes a real-time notification system built with SignalR.
  • Also includes various tree manipulation concepts such as merging nodes, rebasing children, assigning child to node, preventing child node form being orphaned etc.
